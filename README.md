# Tic-Tac-Toe

## Description

Fun game of Tic-Tac-Toe where the first to get three-in-a-row wins!

Play with a friend and have a blast or play against the CPU for a quick relaxing game.

**Features:**

- Modern User Interface
- Intuitive Design
- Custom Markers
- **NEW**: Play Against CPU
